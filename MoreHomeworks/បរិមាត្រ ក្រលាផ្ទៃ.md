## បរិមាត្រ ក្រលាផ្ទៃនែចតុុកោណកែង
### Description
 * សរសេរកម្មវិធីសម្រាប់រកបរិមាត្រ និងក្រឡាផ្ទៃនៃចតុកោណកែងមួយ។
### Input
ចំនួនគត់ $w, h$ ចន្លោះគ្នាមួយដកឃ្លា ដោយដឹងថា $w$ ជាបណ្តោយ និង $h$ ជាទទឹង $0 \le w, h \le 10000$
### Output
  បង្ហាញលទ្ធផលសំណល់នោះ
  * ជួរទី១ ជាបរិមាត្រ
  * ជួរទី២ ជាក្រឡាផ្ទៃ

### Sample
#### input 1
10​ 3
#### output 1
26
30
