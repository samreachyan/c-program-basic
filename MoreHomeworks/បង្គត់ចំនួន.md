## បង្គត់ចំនួន
### Description
  សរសេរកម្មវិធីសម្រាប់បង្គត់តម្លៃមួយចំនួន ដែលតម្លៃតូចជាង 0.5 ត្រូវបង្គត់ចុះ ហើយធំឬស្មើនេះត្រូវបង្គត់ឡើង
### Input
  បញ្ចូលតម្លៃចំនួនទសភាពមួយចំនួន
### Output
  បង្ហាញតម្លៃត្រូវបានបង្គត់

### Sample
#### input 1
4.200939

4

#### input 2
