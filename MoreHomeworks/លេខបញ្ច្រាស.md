# បង្ហាញលេខបញ្ច្រាស
## Decription
គេឱ្យមួយចំនួនគត់ ចូរអ្នកសរសេរកម្មវិធីបង្ហាញលេខបញ្ច្រាស នៃមួយចំនួនដែលបានបញ្ចូលពីក្តារចុច។ 

## Input
  បញ្ចូលចំនួនគត់ $a$
  ដែល $|a| \le 10^{100}$
## Output
បង្ហាញលេខមួយចំនួននោះបញ្រ្ចាសវិញ

## Sample
### input 1
4312
### output 1
2134
